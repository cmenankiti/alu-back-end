# API Project - alu-back-end

## 📌 Description

This project focuses on consuming a REST API and processing data using Python. It demonstrates how to retrieve, manipulate, and export data in different formats such as plain text, CSV, and JSON.

The API used for this project is:
https://jsonplaceholder.typicode.com/

---

## 🎯 Learning Objectives

- How to use REST APIs
- How to make HTTP requests using Python
- How to parse JSON data
- How to export data to CSV and JSON formats
- How to handle command-line arguments in Python
